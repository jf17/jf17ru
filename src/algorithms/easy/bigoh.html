Big Oh

О-большое (Big Oh) - описывает скорость работы алгоритма в худшем случае.



Скорость выполнения алгоритма по увеличению времени выполнения:
  • O(1)
  • O(log2n) (бинарный поиск)
  • O(n) (простой поиск )
  • O(n * log2n) (быстрая сортировка)
  • O(n^2) (сортировка выбором)
  • O(n!) (задача о коммивояжере)
