Дерево решений как классификатор:

Попробуем классифицировать яблоко и апельсин из набора дата сета.
Будем классифицировать по двум параметрам это вес и текстура кожицы.
Устанавливаем Anaconda 5.3 и пишем код Python 3.7. 


from sklearn import  tree // импортируем библиотеку
features = [[140,1],[130,1],[150,0],[170,0]]  // вес и текстура
labels = [0,0,1,1]  //  0 - яблоко , 1 - апельсин 
clf = tree.DecisionTreeClassifier()
clf = clf.fit(features,labels)
print(clf.predict([[150,0]])) //  проверяем выборку
