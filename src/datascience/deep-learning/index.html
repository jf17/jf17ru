Задачи решаемые нейронными сетями:
  • Классификация
    • Для распознавания картинок используют - "свёрточные сети".
    Свёртка - это наложение одной функции на другую.

  • Регрессия
    • Используются "рекурентные сети" (сеть с памятью о предыдущем событии).

  • Кластеризация
    • Обучение с подкреплением.

Тип обучения неронных сетей:
  • С учителем (данные с правильными ответами)
  • Без учителя (данные без правильных ответов)
  • Обучение с подкреплением (Агент получает сигналы от внешней среды)
