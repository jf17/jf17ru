String JavaScript

 Строки - не изменяемый тип данных !!! .
"строка #twitter" // строка в двойных кавычках
'строка текста' // строка в одинарных кавычках

Парсинг строки в Integer.
parseInt(строка , система_исчисления)
Пример:
parseInt( "17" , 10 ) // переведет в десятичное 17
parseInt( "11" , 2 ) // переведет в двоичное 3

Парсинг строки в Float.
parseFloat( строка )

Разбивка строки .
строка.split(',') ; // вернет ['Привет',' мир!']

Реситр символов строки .
строка.toLowerCase() ; // вернет строку с нижнем регистром букв

Длинна строки .
строка.length ; // вернет Number

Удаляем пробельный символ .
строка.trim ; // вернет строку без пробелов

Взятие буквы по индексу .
строка[3] ; // вернет букву

Срез .
строка.slice(0,25) ; // вернет подстроку

Поиск подстроки в строке .
строка.indexOf("#twitter") ; // вернет индекс первого вхождения подстроки в строке
строка.indexOf("#noSubstring") ; // вернет минус один если подстрока не найдена в строке

Начинается ли строка с этой подстроки.
строка.startsWith("#noSubstring") ; // вернет true или false
