JSON

JSON - Текстовый формат обмена данными, основанный на JavaScript.
Ключи (key) в стрковом формате JSON должны быть обязательно в двойных кавычках !!!

метод превращающий обьект в строку (для отправки на сервер):
JSON.stringify({ x: 5, y: 6 }) // expected output: "{"x":5,"y":6}"

метод превращающий строку в обьект ( str -> obj ):
obj = JSON.parse( '{ "x": 5, "y": 6 }' ); // expected output: "{"x":5,"y":6}"
