Callback функция в JavaScript

Callback функция - это функция которая передаётся в другую функцию ввиде параметра(аргумента) и там потом вызывается в нужный момент.

Пример: 
 
/**
* создаём нашу функцию с callback аргументом 
*/
var initSlider = function(callback){

	callback(); // вызываем другую функцию переданную нам в качестве callback'a 
	
}

initSlider(someFunc);  // передаём другую функцию someFunc в качестве аргумета нашей функции
