Install Node JS

 1) Вначале установим PPA для получения доступа к его пакетам.:
curl -sL https://deb.nodesource.com/setup_18.x -o nodesource_setup.sh

2) запустите скрипт с привилегиями sudo
sudo bash nodesource_setup.sh

3) Теперь вы можете установить пакет Node.js:
sudo apt-get install -y nodejs

4) Checked version:
node --version
