MQL:

Список команд MQL для бизнес-объекта. Во всех командах T означает тип, N означает имя, а R означает ревизию бизнес-объекта.

Добавить бизнес-объект :


 add bus T N R policy <policyName> vault <vaultName>

Добавить бизнес-объект и обновить атрибуты в одной команде :


add bus T N R policy <policyName> vault <vaultName> "attribute name 1" "attribute 1 value" "attribute name 2" "attribute 2 value"

Чтобы найти количество бизнес-объектов, используя временную шину запроса :


eval expr ' count true ' on temp query bus "Complaint" * * where " originated > '01/01/2009' ";

Чтобы найти количество бизнес-объектов, которые связаны с бизнес-объектом :


eval expr ' count true ' on expand bus Complaint CPT090000011 1;

Чтобы узнать текущий контекст:


print context;

Чтобы узнать, сколько пользователей подключено к eMatrix (sesssions):


sessions;

Для мониторинга использования памяти :


monitor memory

Узнать версию установленного программного обеспечения :


version;
