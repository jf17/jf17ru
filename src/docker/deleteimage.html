Docker

Удаление образов:

1) Прежде чем удалить image нужно сначала удалить все с ним свазянные контейнеры! 


sudo docker stop $(sudo docker ps -a -q)
sudo docker rm $(sudo docker ps -a -q)



2) После удаления всех контейнеров можно приступать к удалению самого образа ! 


sudo docker rmi id_образа

// посмотреть id всех образов можно командой
sudo docker ps -a
