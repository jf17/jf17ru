PHP

Установка :

// add the PHP PPA repository
sudo add-apt-repository ppa:ondrej/php -y
sudo apt update
sudo apt install php8.3 libapache2-mod-php -y

Настройка версии PHP :

// отключить версию PHP8.1
sudo a2dismod php8.1
// включить версию PHP8.3
sudo a2enmod php8.3
// перезагрузить сервер Apache2
sudo systemctl restart apache2

Проверка версии PHP :

// запустить через sudo su
echo "<?php phpinfo(); ?>" > /var/www/html/info.php
