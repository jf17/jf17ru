Индекс (базы данных)
  • Селективные данные в колонке - данные которые при индексации разбиваются на множество групп
  • Не селективные данные в колонке - данные которые при индексации разбиваются на малое количество групп (пример колонка 'sex' которая разобьётся всего на две группы это 'мужской' и 'женский' )
