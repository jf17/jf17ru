Magic Number (Магическое Число) :

 Антипаттерн использования числовых литералов в коде без объяснения их значения. Числа появляются в коде без контекста, что затрудняет понимание и сопровождение.



Признаки проблемы:

 Числовые константы без имён, повторяющиеся числа в разных местах кода, отсутствие комментариев к значениям.



Как избежать:

 Использование именованных констант, вынесение магических чисел в конфигурационные файлы, добавление комментариев к значениям.



Простой пример из жизни:

 Как рецепт без объяснений: "добавьте 42 грамма специй". Почему именно 42? Что за специи? Через год вы сами не вспомните, что это значило.
