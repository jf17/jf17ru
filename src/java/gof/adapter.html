Adapter (Адаптер) :

 Паттерн позволяет объектам с несовместимыми интерфейсами работать вместе. Адаптер выступает прослойкой между двумя объектами, превращая вызовы одного в вызовы понятные другому.



Для чего используется:

 Интеграция классов с несовместимыми интерфейсами. Использование стороннего кода без изменения существующего.



Пример использования:

 Интеграция с внешними API, адаптеры для legacy кода, обёртки для сторонних библиотек.



Простой пример из жизни:

 Как переходник для розетки: у вас европейская вилка, а розетка американская. Переходник позволяет им работать вместе, не меняя ни вилку, ни розетку.
