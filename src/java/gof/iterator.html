Iterator (Итератор) :

 Паттерн даёт возможность последовательно обходить элементы составных объектов, не раскрывая их внутреннего представления.



Для чего используется:

 Обход коллекций без раскрытия их внутренней структуры. Унификация интерфейса обхода разных структур данных.



Пример использования:

 Интерфейс Iterator в Java Collections Framework, foreach циклы.



Простой пример из жизни:

 Как перелистывание страниц книги: вы читаете страницу за страницей, не зная общего количества страниц и внутренней структуры книги.
