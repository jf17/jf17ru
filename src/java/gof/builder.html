Builder (Строитель) :

 Паттерн Builder позволяет создавать сложные объекты пошагово. Строитель даёт возможность использовать один и тот же код строительства для получения разных представлений объектов.



Для чего используется:

 Создание объектов с множественными необязательными параметрами. Альтернатива конструкторам с большим количеством параметров.



Пример использования:

 StringBuilder, создание объектов конфигурации, построение SQL-запросов.



Простой пример из жизни:

 Как заказ бургера в McDonald's: вы выбираете основу (чизбургер), добавляете соус, убираете лук, добавляете бекон. Каждый шаг строит финальный продукт пошагово.
