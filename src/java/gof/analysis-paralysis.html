Analysis Paralysis (Паралич Анализа) :

 Антипаттерн чрезмерного анализа и планирования, который препятствует принятию решений и началу разработки. Команда застревает в бесконечном анализе вместо действий.



Признаки проблемы:

 Бесконечные совещания без решений, избыточное планирование, страх принятия неправильного решения, отсутствие прогресса в разработке.



Как избежать:

 Установка временных рамок для анализа, итеративная разработка, принятие решений на основе доступной информации, прототипирование.



Простой пример из жизни:

 Как выбор ресторана для ужина: изучаете отзывы, меню, цены, местоположение часами, а в итоге так и не идёте никуда, потому что не можете решить.
