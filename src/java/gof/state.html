State (Состояние) :

 Паттерн позволяет объектам менять поведение в зависимости от своего состояния. Извне создаётся впечатление, что изменился класс объекта.



Для чего используется:

 Изменение поведения объекта в зависимости от его внутреннего состояния. Избежание больших условных операторов.



Пример использования:

 Конечные автоматы, игровые персонажи с разными состояниями, медиаплееры (воспроизведение, пауза, стоп).



Простой пример из жизни:

 Как торговый автомат: в зависимости от состояния (ожидание монет, выбор товара, выдача сдачи) он ведёт себя по-разному на одни и те же действия.
