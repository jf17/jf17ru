Template Method (Шаблонный Метод) :

 Паттерн определяет скелет алгоритма в суперклассе, позволяя подклассам переопределять отдельные шаги алгоритма, не изменяя его общей структуры.



Для чего используется:

 Определение общего алгоритма с возможностью изменения отдельных шагов в подклассах. Избежание дублирования кода.



Пример использования:

 Фреймворки с lifecycle методами, алгоритмы сортировки с настраиваемым сравнением, обработка данных с разными форматами.



Простой пример из жизни:

 Как рецепт приготовления чая: общие шаги одинаковы (вскипятить воду, заварить, настоять), но детали могут отличаться (тип чая, время заваривания).
