Flyweight (Приспособленец) :

 Паттерн позволяет вместить бóльшее количество объектов в отведённую оперативную память. Приспособленец экономит память, разделяя общее состояние объектов между собой.



Для чего используется:

 Экономия памяти при работе с большим количеством мелких объектов. Разделение внутреннего и внешнего состояния объектов.



Пример использования:

 String Pool в Java, символы в текстовых редакторах, частицы в играх, кэширование объектов.



Простой пример из жизни:

 Как библиотека: одну книгу могут читать много людей по очереди, вместо того чтобы каждый покупал свой экземпляр. Экономия ресурсов через совместное использование.
