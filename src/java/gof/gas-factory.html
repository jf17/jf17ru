Gas Factory (Газовый Завод) :

 Антипаттерн чрезмерного усложнения простых задач. Создание излишне сложной архитектуры там, где можно обойтись простым решением.



Признаки проблемы:

 Избыточная абстракция, множество слоёв для простых операций, переусложнённые паттерны проектирования, код сложнее задачи.



Как избежать:

 Принцип KISS (Keep It Simple, Stupid), оценка сложности задачи перед проектированием, постепенное усложнение по мере необходимости.



Простой пример из жизни:

 Как построить целый завод для производства одного гвоздя: огромные инвестиции, сложная логистика, множество процессов - а нужен всего один гвоздь.
