Visitor (Посетитель) :

 Паттерн позволяет создавать новые операции, не изменяя классы объектов, над которыми эти операции могут выполняться.



Для чего используется:

 Добавление новых операций к существующим классам без их изменения. Разделение алгоритмов от структур данных.



Пример использования:

 Обход и обработка AST (абстрактного синтаксического дерева), экспорт данных в разные форматы, валидация сложных структур.



Простой пример из жизни:

 Как экскурсовод в музее: он знает, как работать с разными экспонатами (картины, скульптуры, артефакты), но сами экспонаты не меняются.
