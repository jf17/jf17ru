Facade (Фасад) :

 Паттерн предоставляет простой интерфейс к сложной подсистеме, содержащей множество взаимосвязанных объектов.



Для чего используется:

 Упрощение интерфейса к сложной подсистеме. Уменьшение зависимостей между клиентом и сложной системой.



Пример использования:

 API библиотек, сервисные слои в приложениях, упрощённые интерфейсы к сложным фреймворкам.



Простой пример из жизни:

 Как стойка регистрации в отеле: вы просто говорите "хочу заселиться", а сотрудник координирует работу с бронированием, уборкой, охраной, бухгалтерией.
