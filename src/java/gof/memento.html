Memento (Хранитель) :

 Паттерн позволяет сохранять и восстанавливать прошлые состояния объектов, не раскрывая подробностей их реализации.



Для чего используется:

 Сохранение снимков состояния объекта для последующего восстановления. Реализация функции отмены (Undo).



Пример использования:

 Функция отмены в текстовых редакторах, сохранение состояния игры, откат транзакций в базах данных.



Простой пример из жизни:

 Как сохранение игры в видеоиграх: вы можете сохранить текущий прогресс и позже вернуться к этому моменту, если что-то пошло не так.
