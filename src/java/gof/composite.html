Composite (Компоновщик) :

 Паттерн позволяет сгруппировать множество объектов в древовидную структуру, а затем работать с ней так, как будто это единичный объект.



Для чего используется:

 Представление иерархий объектов типа "часть-целое". Единообразная работа с отдельными объектами и их композициями.



Пример использования:

 Файловые системы (файлы и папки), GUI компоненты (контейнеры и элементы), математические выражения.



Простой пример из жизни:

 Как организационная структура компании: и отдельный сотрудник, и целый отдел можно рассматривать одинаково - у них есть бюджет, задачи, руководитель.
