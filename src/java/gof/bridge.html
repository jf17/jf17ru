Bridge (Мост) :

 Паттерн разделяет один или несколько классов на две отдельные иерархии — абстракцию и реализацию, позволяя изменять их независимо друг от друга.



Для чего используется:

 Разделение абстракции и реализации для их независимого изменения. Избежание постоянной привязки абстракции к реализации.



Пример использования:

 Драйверы устройств, графические API для разных платформ, абстракции баз данных.



Простой пример из жизни:

 Как пульт от телевизора: один пульт (абстракция) может управлять разными телевизорами (реализации) - Samsung, LG, Sony. Пульт и телевизор развиваются независимо.
