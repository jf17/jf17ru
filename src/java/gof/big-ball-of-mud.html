Big Ball of Mud (Большой Ком Грязи) :

 Антипаттерн, описывающий систему без чёткой архитектуры. Код представляет собой хаотичную структуру, где компоненты тесно связаны и плохо организованы.



Признаки проблемы:

 Отсутствие чёткой архитектуры, тесная связанность компонентов, сложность понимания и модификации кода, отсутствие документации.



Как избежать:

 Планирование архитектуры, применение принципов SOLID, рефакторинг, документирование кода, code review.



Простой пример из жизни:

 Как захламлённая квартира: вещи лежат где попало, найти что-то сложно, убрать одну вещь - значит нарушить всю "систему". Никто не помнит, зачем что лежит.
