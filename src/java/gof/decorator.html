Decorator (Декоратор) :

 Паттерн позволяет динамически добавлять объектам новую функциональность, оборачивая их в полезные «обёртки». Декоратор предоставляет гибкую альтернативу наследованию для расширения функциональности.



Для чего используется:

 Динамическое добавление поведения объектам без изменения их структуры. Альтернатива множественному наследованию.



Пример использования:

 Java I/O streams (BufferedReader, FileReader), UI компоненты с дополнительными возможностями.



Простой пример из жизни:

 Как одевание зимой: базовая футболка, сверху свитер, затем куртка. Каждый слой добавляет функциональность (тепло), не меняя основу.
