Java Collections Framework.

 картинка Java Collections Framework

Главные Интерфейсы:
  • Iterable
  • Collection
  • List (список)
  • Set (для хранения множеств уникальных объектов)
  • Queue (очередь)
  • Map (key-value)
  • Map.Entry

Расширенные Интерфейсы:
  • Iterator
  • SortedMap
  • NavigableMap
  • SortedSet (сортированный Set)
  • NavigableSet(поиск по соответствию)
  • Deque(двунаправленная очередь)

Классы:
  • ArrayList
  • LinkedList
  • HashMap
  • LinkedHashMap
  • HashTable устаревший (legacy)
  • TreeMap
  • HashSet (Set из хеш-кодов)
  • LinkedHashSet
  • TreeSet
  • ArrayDeque
  • PriorityQueue
  • Vector устаревший (legacy)

Абстрактные классы:
  • AbstractCollection
  • AbstractList
  • AbstractMap
  • AbstractSet
  • AbstractQueue
