Главная

Java

Java Collections Framework.

Главные Интерфейсы:

  • Iterable
  • Collection
  • List (список)
  • Set (для хранения множеств уникальных объектов)
  • Queue (очередь)
  • Map (key-value)

Расширенные Интерфейсы:

  • Deque(двунаправленная очередь)
  • SortedSet (сортированный Set)
  • NavigableSet(поиск по соответствию)
  • Интерфейс SortedMap
  • Интерфейс NavigableMap

Классы:

  • ArrayList
  • LinkedList
  • ArrayDeque
  • HashSet (Set из хеш-кодов)
  • TreeSet
  • LinkedHashSet
  • PriorityQueue
  • HashMap
  • TreeMap
