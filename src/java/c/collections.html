Java Collections Framework.

 картинка Java Collections Framework

Набор статических методов для работы с коллекциями

  • Collections класс
  • Generics обобщеный тип .

Главные Интерфейсы:
  • Iterable
  • Collection
  • List (список)
  • Set (для хранения множеств уникальных объектов)
  • Queue (очередь)
  • Map (key-value)
  • Map.Entry

Расширенные Интерфейсы:
  • Iterator
  • SortedMap
  • NavigableMap
  • SortedSet (сортированный Set)
  • NavigableSet(поиск по соответствию)
  • Deque(двунаправленная очередь)

List:
  • ArrayList
  • LinkedList — рекомендуется не использовать
  • Stack
  • Vector устаревший (legacy)
  • Collections.synchronizedList( * )

Map:
  • HashMap
  • LinkedHashMap
  • HashTable устаревший (legacy)
  • TreeMap

Set:
  • HashSet (Set из хеш-кодов)
  • LinkedHashSet
  • TreeSet

Queue:
  • ArrayDeque
  • PriorityQueue

Абстрактные классы:
  • AbstractCollection
  • AbstractList
  • AbstractMap
  • AbstractSet
  • AbstractQueue
