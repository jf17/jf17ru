интерфейс Cloneable.

package java.lang.*

Вызов метода clone объекта на экземпляре, который не реализует Cloneable интерфейс приводит к возникновению исключения CloneNotSupportedException.
По соглашению классы, которые реализуют этот интерфейс, должны переопределить Object.clone (который защищен) открытым методом. Смотрите Object.clone() для получения подробной информации о переопределении этого метода.
Обратите внимание, что этот интерфейс не содержит метод clone. Поэтому невозможно клонировать объект только в силу того, что он реализует этот интерфейс. Даже если метод clone вызывается рефлективно, нет никакой гарантии, что он будет успешным.
