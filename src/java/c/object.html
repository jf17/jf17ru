Object.

package java.lang.*

Объект – это экземпляр класса или массив.
Значения ссылок (чаще просто ссылки) – это указатели на эти объекты, и специальная ссылка null, которая означает отсутствиет объекта.


методы Object:
  • String *.toString() : Метод toString служит для получения представления данного объекта в виде строки.
    getClass().getName() + '@' + Integer.toHexString(hashCode()) == Person@7960847b . Mетод toString() нередко переопределяют.

  • int *.hashCode() : Метод hashCode позволяет задать некоторое числовое значение, которое будет соответствовать данному объекту или его хэш-код.Если два объекта равны по методу equals (Object), то вызов метода hashCode для каждого из двух объектов должен привести к одному целочисленному результату.
    1) для одного и того-же объекта, хеш-код всегда будет одинаковым;
    2) если объекты одинаковые, то и хеш-коды одинаковые (но не наоборот, см. правило 3).
    3) если хеш-коды равны, то входные объекты не всегда равны (коллизия);
    4) если хеш-коды разные, то и объекты гарантированно разные;

  • Class<?> *.getClass() : Метод getClass позволяет получить тип данного объекта .

  • boolean *.equals() : Метод equals по умолчанию сравнивает ссылки. return (this == obj); Mетод equals() нередко переопределяют.
