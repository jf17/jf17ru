JAVAC

Javac — оптимизирующий компилятор языка java, включенный в состав многих Java Development Kit (JDK).

1) Создаем(вручную) папку classes (куда будут сохраняться результаты компиляции) : 

-d classes

2) создаём строку CLASSPATH (откуда будут браться сторонние библиотеки) :
для Linux вместо точки с зяпятой используется двоеточие и наклон в другую сторону. 

-cp "testzip\zlib\autocomplete-2.6.1.jar;testzip\zlib\rsyntaxtextarea-2.6.1.jar"

3) добавляем все папки с *.java файлами 

java\ru\jf17\ide\gui\*.java

4) запускаем 

javac -d classes -cp "testzip\zlib\autocomplete-2.6.1.jar;testzip\zlib\rsyntaxtextarea-2.6.1.jar" java\ru\jf17\ide\gui\*.java
