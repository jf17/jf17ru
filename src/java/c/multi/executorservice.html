Многопоточность

ExecutorService. execute

 // когда нужно тупо чтото выполнить 

ExecutorService executorService = Executors.newFixedThreadPool(10);

executorService.execute(new Runnable() {
    public void run() {
        System.out.println("Asynchronous task");
    }
});

executorService.shutdown();



ExecutorService. submit

 // когда нужно вернуть значение 

ExecutorService executor = Executors.newFixedThreadPool(10);

 Future<Integer> futureTask1 = executor.submit(() -> {
            System.out.println("I'm Callable task.");
            return 1 + 1;
           });

Integer result = futureTask1.get(5, TimeUnit.SECONDS);

executor.shutdown();



методы ExecutorService:
  • void *.shutdown() : инициирует упорядочное завершение, новые задачи не принимаются . Этот метод не ожидает выполнение ранее отправленных задач !

  • void *.shutdownNow() : Пытается остановить все активные задачи, новые задачи не принимаются . Этот метод не ожидает выполнение ранее отправленных задач !

  • boolean *.awaitTermination(long timeout,TimeUnit unit) : блокируется до тех пор, пока все задачи не завершат выполнение после запроса на выключение, или не истечет время ожидания, или текущий поток не прервется.
