Executors.

package java.util.concurrent.*

появился в java 1.5


методы Executors:
  • ExecutorService Executors.newSingleThreadExecutor() : возвращает один рабочий поток . Задачи гарантированно выполняются последовательно.

  • ExecutorService Executors.newFixedThreadPool(int nThreads) : возвращает фиксированное кол-во рабочих потоков

  • ExecutorService Executors.newCachedThreadPool() : создает новые потоки по мере необходимости .

  • ScheduledExecutorService Executors.newScheduledThreadPool(int corePoolSize) : создает пул потоков, который может запланировать выполнение команд после задержки или переодическое выполнение.
