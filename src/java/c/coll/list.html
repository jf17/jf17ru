Collections Framework

интерфейс List(список):

Список это упорядоченная в порядке добавления коллекция, так же известная как последовательность (sequence). Дублирующие элементы в большинстве реализаций — разрешены. Позволяет доступ по индексу элемента. Расширяет Collection интерфейс.

имплементация в классах:
  • ArrayList
  • LinkedList

методы интерфейса List:
  • boolean *.add(E e)- добавляет новый обьект в конец коллекции .
  • E *.get(int index) - получить обьект по индексу.
  • E *.remove(int index) - удалить обьект по индексу .
  • boolean *.remove(Object o) - удалить обьект по ссылке .
  • boolean *.isEmpty() - пустая ли коллекция ?
  • void *.clear() - очистить коллекцию .
  • boolean *.contains(Object o) - содержит ли коллекция этот обьект ?
  • int *.size() - размер коллекции
  • Object[] *.toArray() - преобразовать коллекцию в массив .
  • List *.subList(int fromIndex, int toIndex) - вернуть срез коллекции .
  • void *.sort(Comparator c) - отсортировать коллекцию.
  • Iterator *.iterator() - Возвращает итератор чтобы можно было пройти по коллекции .
