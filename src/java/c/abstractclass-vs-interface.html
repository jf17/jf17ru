Interface Абстрактный класс
Поля всегда являются public static final (константы) Может содержать переменные экземпляра с различными модификаторами доступа (например, private, protected).
До Java 8 содержал только абстрактные методы, но начиная с Java 8, поддерживает default и static методы.
Начиная с Java 9, можно использовать private методы для внутренних нужд. 		Методы могут быть public, protected, private или package-private.
Конструкторы отсутствуют. Может иметь конструкторы.
Реализуется с помощью ключевого слова implements. Наследуется с помощью ключевого слова extends.
Класс может реализовывать несколько интерфейсов. Класс может наследовать только один абстрактный класс.
