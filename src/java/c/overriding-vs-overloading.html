Overriding (Переопределение):

Overriding — это процесс, при котором подкласс предоставляет собственную реализацию метода, который уже определён в его суперклассе.

Пример: 
    class Animal {
        void sound() {
            System.out.println("Animal makes a sound");
        }
    }

    class Dog extends Animal {
        @Override
        void sound() {
            System.out.println("Dog barks");
        }
    }


Overloading (Перегрузка):

Overloading — это создание нескольких методов с одним и тем же именем, но с разной сигнатурой (количество или тип параметров).

Пример: 
    class Calculator {
        int add(int a, int b) {
            return a + b;
        }

        double add(double a, double b) {
            return a + b;
        }

        int add(int a, int b, int c) {
            return a + b + c;
        }
    }



Особенность Overriding Overloading
Сигнатура Должна быть идентичной Должна быть разной
Возвращаемый тип Должен совпадать или быть ковариантным Может отличаться
Аннотация @Override Используется Не используется
Полиморфизм Поддерживает Не поддерживает
Время определения Время выполнения (runtime) Время компиляции (compile-time)
