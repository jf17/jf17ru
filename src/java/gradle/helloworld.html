Gradle . Hello World

 

1)создаём папку проекта 
2)переходим в папку 
3) gradle init --type java-application
4) добавляем в файл build.gradle строчки :
jar {
   manifest {
      attributes 'Main-Class': 'App'
   }
}

5) gradle clean
6) gradle build

7) в папке my-test-java-app\build\libs\ будет лежать ваш *.jar файл .
