настройка autodeploy WAR в Intellij Idea:

  1. скачиваем сервер TomCat
  2. в IDEA заходим в File->Settings -> Application Servers
  3. жмём на плюсик и добавляем TomCat Server (выбираем папку где лежит сервак)
  4. add Configuration -> жмём плюсик и добавляем TomCat Server local
  5. в поле Name указываем имя проекта
  6. переходим во вкладку Deployment
  7. жмём плюсик выбираем Artifact
  8. выбираем артифакт с именем содержащим exploded
