HelloWorld сервлет в Intellij IDEA:
  • Java Enterprise -> Web Application
  • создаём класс HelloServlet.java и регистрируем сервлет через аннотацию @WebServlet("/hello") :
    • 
import java.io.PrintWriter;
import java.io.IOException;
import javax.servlet.ServletException;
import javax.servlet.annotation.WebServlet;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;
 
@WebServlet("/hello")
public class HelloServlet extends HttpServlet {
 
    protected void doGet(HttpServletRequest request, HttpServletResponse response) throws ServletException, IOException {
         
        response.setContentType("text/html");
        PrintWriter writer = response.getWriter();
        try {
            writer.println("<h2>Hello from HelloServlet</h2>");
        } finally {
            writer.close();  
        }
    }
}
  • или регистрируем сервлет через файл web.xml :
    • 
    <servlet>
        <servlet-name>HelloServlet</servlet-name>
        <servlet-class>ru.jf17.demo.HelloServlet</servlet-class>
    </servlet>
    <servlet-mapping>
        <servlet-name>HelloServlet</servlet-name>
        <url-pattern>/hello</url-pattern>
   </servlet-mapping>
  • открываем результат
