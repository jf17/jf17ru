GlassFish:

Перед первым запуском нужно поменять файл настроек (добавить расположение JDK)

  • зайти GlassFish\glassfish5\glassfish\config\asenv.bat
  • в конец файла добавить строчку "set AS_JAVA=C:\Program Files\Java\jdk1.8.0_151"
  • запуск E:\GlassFish\glassfish5\bin>asadmin.bat start-domain
  • остановка E:\GlassFish\glassfish5\bin>asadmin.bat stop-domain
