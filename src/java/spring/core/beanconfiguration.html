@Bean + @Configuration :

 Используется для настройки контекста через Код !!!

ApplicationContext context = new AnnotationConfigApplicationContext(MessageConfig.class);
Massage message = context.getBean(Message.class)



@Configuration - Прописывается перед классом , означает что класс может быть использован контейнером Spring IoC как конфигурационный класс для бинов

@Bean - Прописывается перед методом , информирует Spring о том, что возвращаемый данным методом обьект должен быть зарегистрирован , как бин.
