AnnotationConfigApplicationContext

  • Класс который позволяет создать конфигурацию приложение через Аннотации (вместо XML-файла)
  • Класс AppContext должен содержать аннотации @Configuration и @Bean


    • Регистрация через конструктор : 
    public static void main(String[] args){
    ApplicationContext ctx = new AnnotationConfigApplicationContext(AppContext.class);
}

    Регистрация через метод : 
    public static void main(String[] args){
    ApplicationContext ctx = new AnnotationConfigApplicationContext();
	ctx.register(AppContext.class);
}
