OpenCV 4 установка на Linux.
  • качаем библиотеку OpenCV

  • sudo apt-get update
  • sudo apt-get install build-essential
  • sudo apt-get install cmake

    • # Предполагается, что мы в каталоге, куда был распакован архив
  • mkdir release
  • cd release
  • cmake -D CMAKE_BUILD_TYPE=RELEASE -D CMAKE_INSTALL_PREFIX=/home/alex/OpenCV/ ../ # куда будет установлена библиотека после компиляции

  • make # сама компиляция (очень долгий процес)
  • sudo make install # установка
